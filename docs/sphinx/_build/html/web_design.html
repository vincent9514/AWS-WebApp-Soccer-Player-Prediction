Web Design¶

Index Page¶

This is the index page. User can click “Wanna Predict?” to make prediction. In addition, user can click other buttons for reference.

Search Page¶

This is the seach page where user can input the player’s attributes and make prediction.

Prediction Result¶

This page shows the prediction result.

Click Database¶

When user clicks the “FIFA18 Database”, user can browse the FIFA18 Offical Database.

Click About-us¶

When user click “About us”, the FIFA offical introduction page will be shown.

Click Model¶

When user click “model” button, the kaggle competition page will be shown.

Click Impact¶

When user click “Impact”, the relative article will be shown.

Click Connect¶

When user click “Connect”, the developer’s introduction page will be shown.

Fill_Warning¶

User has to input all attributes, otherwise the warning will be shown.

Bottom¶

This is the bottom page.
